## lg-inner-focus-outline

Sets the focus outline inside the element, by using multiple box shadows.

### Arguments

| Name        | Description                         |           Default           | Required |
| ----------- | ----------------------------------- | :-------------------------: | :------: |
| `$bg-color` | The background color of the element |             n/a             |   Yes    |
| `$color`    | The colour of the outline           | `var(--colour-greyscale-0)` |    No    |

## lg-focus-outline

Sets the focus outline outside the element, by using box shadows. This is currently used on buttons.

### Arguments

| Name              | Description                        |              Default               | Required |
| ----------------- | ---------------------------------- | :--------------------------------: | :------: |
| `$bg-color`       | The outer colour of the box-shadow |    `var(--default-focus-color)`    |    No    |
| `$inner-bg-color` | The inner colour of the box-shadow | `var(--default-inner-focus-color)` |    No    |

## lg-link

Styles elements with the link style.

### Arguments

| Name               | Description                                 |            Default            | Required |
| ------------------ | ------------------------------------------- | :---------------------------: | :------: |
| `$default-color`   | The default colour of the element           |      `var(--link-color)`      |    No    |
| `$hover-color`     | The hover colour of the element             |   `var(--link-hover-color)`   |    No    |
| `$visited-color`   | The visited colour of the element           |  `var(--link-visited-color)`  |    No    |
| `$active-color`    | The active colour of the element            |  `var(--link-active-color)`   |    No    |
| `$active-bg-color` | The active background colour of the element | `var(--link-active-bg-color)` |    No    |
| `$focus-color`     | The focus colour of the element             |   `var(--link-focus-color)`   |    No    |
| `$focus-bg-color`  | The focus background colour of the element  | `var(--link-focus-bg-color)`  |    No    |

## lg-unstyled-link

Revert the links styles added with `lg-link`.

## lg-font-size

Styles text to one of the predefined font sizes. '0-8' and '0-6' are the sub body font sizes.

### Arguments

| Name        | Description                                                          |                  Default                  | Required |
| ----------- | -------------------------------------------------------------------- | :---------------------------------------: | :------: |
| `$size`     | The font size (e.g. `7`, `6`, `5`, `4`, `3`, `2`, `1`, `0-8`, `0-6`) |                    n/a                    |   Yes    |
| `$modifier` | Font weight modifier, (e.g. `300`, `400`, `500`, `700`)              | `400` for Nunito Sans, `500` for ABC Otto |    No    |
| `$type`     | Font family type (e.g. 'expressive')                                 |                   null                    |    No    |

## lg-visually-hidden

Provides styles to hide information intended only for screen readers from the layout of the rendered page.

## lg-breakpoint

Allows breakpoints to be added to css blocks.

### Arguments

| Name          | Description                                     | Default | Required |
| ------------- | ----------------------------------------------- | :-----: | :------: |
| `$breakpoint` | The string value for the breakpoint screen size |   n/a   |   Yes    |

## lg-variant

Styles components and native child elements to the specified variant.

### Arguments

| Name       | Description                                                              |  Default  | Required |
| ---------- | ------------------------------------------------------------------------ | :-------: | :------: |
| `$variant` | The variant type (e.g. `generic`, `info`, `success`, `warning`, `error`) | `generic` |    No    |
