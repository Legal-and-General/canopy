# Data point

<CustomBadge text="Pending" /> <span class="small-text">Due for brand modernisation</span>

<p class="standfirst">Display data organised by heading and value. A data point can be used on its own, or grouped into a data point list where each one will be displayed horizontally and given a list-item role.</p>

## Usage

Import the data point components in your application:

```js
@NgModule({
  ...
  imports: [ ..., LgDataPointComponent, etc. ],
});
```

In your HTML, for a single data point:

<Source of={DataPointStories.SingleDataPoint} />

and for a data point list:

<Source of={DataPointStories.DataPointsListStory} />

### Inputs

| Name           | Description                        |   Type   | Default | Required |
|----------------|------------------------------------|:--------:|:-------:|:--------:|
| `headingLevel` | Allows the heading level to be set | `number` |   n/a   |   Yes    |
