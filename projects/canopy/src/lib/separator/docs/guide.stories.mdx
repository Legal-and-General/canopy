# Separator

<p class="standfirst">Separators are used to visually divide sections of content.</p>

![example](docs/separator/example.png)

## Usage

Use this component when you want to split your content into clear sections or groupings.

Import the separator component into your application:

```js
@NgModule({
    ...
    imports: [ ..., LgSeparatorComponent ],
});
```



and in your HTML:

```html
<lg-separator [variant]="variant" [hasRole]="hasRole"></lg-separator>
```

## How it works

The separator component can be positioned at either full or partial width to help define distinct sections or groups within the page, a container or body content.

## Additional development details

### Inputs

| Name      | Description                                          |        Type        | Default | Required |
|-----------|------------------------------------------------------|:------------------:|:-------:|:--------:|
| `variant` | The variant of separator: `solid`, `dotted`          | `SeparatorVariant` | `solid` |    No    |
| `hasRole` | If true, adds a role of `separator` to the component |     `boolean`      | `false` |    No    |
