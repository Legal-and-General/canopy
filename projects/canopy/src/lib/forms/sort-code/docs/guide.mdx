# Sort code

<CustomBadge text="Pending" style={{ backgroundColor: '#005dba', color: '#fff' }} /> <span class="small-text">Due for brand modernisation</span>

<p class="standfirst">Provides a directive to apply formatting and specific validators to the input component for entering a sort code.</p>

## Usage

Import the sort code and the [text input component](./?path=/docs/components-forms-text-input-guide--docs) into your application:

```js
  imports: [ ..., LgSortCodeDirective, LgInputComponent, LgHintComponent ],
```

and in your HTML:

<Source of={SortCodeStories.SortCode} />

Both the `lgInput` and `lgSortCode` directives have to be present on the input element.

The exact `lg-hint` should be copied as well to ensure the correct context is provided to the user.

**We recommend using <a href="https://angular.io/guide/reactive-forms" target="_blank">Angular's reactive forms</a>.**

### Validators

The sort code directive automatically adds the `required` and `pattern` validators by default.
The `pattern` validator checks that the sort code value has the following patterns: `000000`, `00 00 00` or `00-00-00`.

See the [form validation guide](./?path=/docs/components-forms-form-validation-guide--docs) for additional information.

### Format of the control value

On blur of the input we auto-format the value of the control to `00-00-00`.
