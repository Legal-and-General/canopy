# Heading

<p class="standfirst">Allows for headings to be set dynamically.</p>

## Usage

**This component should only be used within other components (see the <a href="https://github.com/Legal-and-General/canopy/blob/master/projects/canopy/src/lib/accordion/accordion-panel-heading/accordion-panel-heading.component.html" target="_blank">accordion component</a> as an example).**

Import the heading module in your application:

```js
@NgModule({
  ...
  imports: [ ..., LgHeadingModule ],
});
```

### Inputs

| Name    | Description       |   Type   |   Default   | Required |
|---------|-------------------|:--------:|:-----------:|:--------:|
| `level` | The heading level | `number` | `undefined` |   Yes    |
