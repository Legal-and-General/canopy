# Heading

This component allows for headings to be set dynamically.

**It should only be used within other components (see the [accordion component](https://github.com/Legal-and-General/canopy/blob/master/projects/canopy/src/lib/accordion/accordion-panel-heading/accordion-panel-heading.component.html) as an example).**

## Usage

Import the heading module in your module:

~~~js
@NgModule({
  ...
  imports: [ ..., LgHeadingModule ],
});
~~~

### Inputs

| Name | Description | Type | Default | Required |
|------|-------------|:----:|:-----:|:-----:|
| `level` | The heading level | number | `undefined` | Yes |
