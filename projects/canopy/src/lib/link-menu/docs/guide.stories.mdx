# Link menu

<p class="standfirst">Link menu is used to list an array of links.</p>

## Usage

Import the link menu module in your application:

```js
@NgModule({
  ...
  imports: [ ..., LgLinkMenuModule ],
});
```

and in your HTML:

<Source id="components-link-menu-examples--standard-link-menu"></Source>


## Additional development details

### LgLinkMenuComponent
This is where the links are projected.

### LgLinkMenuItemComponent
This is where the `LgLinkMenuItemHeadingComponent` and `LgLinkMenuItemContentComponent` of the link are projected and positioned.

Depending on whether the parent, `<a>` element has `target="_blank"`, i.e. attempts to open in a new tab, the icon displayed will
differ.

### LgLinkMenuItemHeadingComponent
This is the main label of the link.

### LgLinkMenuItemContentComponent
This is used to add a description for the link.
