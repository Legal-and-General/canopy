# Shadow

This directive adds a box shadow to a component.

## Usage

Import the shadow module in your application:

```js
@NgModule({
  ...
  imports: [ ..., LgShadowModule ],
});
```

and in your HTML:

<Source id="helpers-directives-shadow-examples--shadow-story"></Source>

### Inputs
| Name       | Description                                     |   Type    | Default | Required |
|------------|-------------------------------------------------|:---------:|:-------:|:--------:|
| `lgShadow` | Whether the element should have a shadow or not | `boolean` | `null`  |   Yes    |
