# Shadow

<CustomBadge text="Awaiting deprecation" style={{ backgroundColor: '#ffb600', borderColor: '#ffb600', color: '#1d1d1b' }} />

<p class="standfirst">This directive adds a box shadow to a component.</p>

## Usage

Import the shadow directive in your application:

```js
@NgModule({
  ...
  imports: [ ..., LgShadowDirective ],
});
```

and in your HTML:

<Source of={ShadowStories.Shadow} />

### Inputs

| Name            | Description                                                           |   Type    | Default | Required |
|-----------------|-----------------------------------------------------------------------|:---------:|:-------:|:--------:|
| `lgShadow`      | Whether the element should have a shadow or not                       | `boolean` | `null`  |   Yes    |
| `hasHoverState` | Whether the element should have a deeper shadow on hover state or not | `boolean` | `false` |   No     |
