# Focus

This directive allows for focus to be set dynamically on a focusable element.

## Usage

Import the focus directive in your application:

```
@NgModule({
  ...
  imports: [ ..., LgFocusDirective ],
});
```

and in your HTML:

<Source of={FocusStories.focus} />

### Inputs

| Name      | Description                                  |   Type    | Default | Required |
|-----------|----------------------------------------------|:---------:|:-------:|:--------:|
| `lgFocus` | Whether the element should be focused or not | `boolean` | `null`  |   Yes    |
