# Button

Provides styles and behaviours for Canopy buttons. `lg-button` is a directive which can be applied to any `<button>` or `<a>` element.

Import the button module in your module:
~~~js
@NgModule({
  ...
  imports: [ ..., LgButtonModule ],
});
~~~
